# Приложение для управления приютом для животных

### Состоит из двух микросервисов

* Бекенд часть со всей логикой приложения
* Фронтенд часть - телеграмм бот для пользователей

### Над приложением работали

* Елена Матвиенко
* Владимир Иваненко
* Сергей Голубцов
* Марсель Мингарипов


## Реализован следующий функционал
+ бронирование животного на срок 24 часа. По истечении срока бронь снимается
